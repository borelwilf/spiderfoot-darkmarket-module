# sfp_darkmarket

SpiderFoot-Modul zur Extraktion und Korrelation technischer
Identifikationsmerkmale (PGP-Schlüssel, Kryptowährungsadressen,
E-Mail-Adressen, Benutzernamen, Kontaktinformationen) aus dem
Inhalt von Darknet-Marktplätzen.

Entwickelt im Rahmen der Masterarbeit von Borel Wilfried Nangse
Tchantchou, Hochschule Niederrhein, 2026.

## Installation
Datei in den SpiderFoot-Modulordner kopieren:
modules/sfp_darkmarket.py
